    Returns an object containing the given headers as well as other headers (like the authorization header and custom headers) that should be included in a request.

    Parameters

    • callerHeaders: null | object

      headers that the caller wishes to include in the request. This method will not override these headers with its own. Thus, if all the headers that this method was planning to add are already specified here, this method will return an object with exactly the same headers.

    • accessToken: string

      the access token that will be used to make the request

    Returns Record<string, string>

    • a new object with the headers needed for the request
